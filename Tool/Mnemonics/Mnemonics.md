## 词组联结

每天做 30 对抽象词汇的联结，不断增加数量，坚持一两个月

>图像联接

```
建设、负面
建设一栋楼(豆腐渣工程)，一旦倒塌就会出现负面新闻

免疫、支付
去医院打乙肝疫苗(免疫作用)，支付医药费

简单、启示
煎蛋塞进骑士嘴里

无奈、合理
盒子里(合理)装着很多的乌黑的牛奶瓶(无奈=乌黑的牛奶瓶)
```

> 逻辑联结

```
干净、技术
洗干净鸡肉后，把鸡肉煮熟(熟鸡=技术)

散开、平台
桌球杆一打，很多球被击中后散开在桌球平台上

证明、切换、置顶、收集、安置
电工师傅上门修灯泡他出示工作证件证明身份后，我开门让他进来。电工师傅把旧灯泡取出来，换上(切换)新灯泡新灯泡安装在房门的顶部(置顶)。电工师傅收集(收集)好自己所有的工具，把它们安置(安置)在工具箱的卡槽内。

领导、决策损失、矛盾、应对
一般来说，领导(领导)的决策(决策)不是带来利益，就是带来损失(损失)，如果造成的经济损失比较大，公司陷入财务危机，就发不起工资，员工就会和领导发生矛盾(矛盾 )冲突,外部势力也会趁虚而入。如果没有方案应对(应对)，公司就会倒闭。

```

## 定桩

```
电子商务法的八大亮点
桩子：鼠标
手放在鼠标上，控制计算机上网购物，鼠标下面有红色的亮点(电子商务=上网购物，八=巴掌，亮点=鼠标的红外线灯)。

促进发展，严格范围
桩子：键盘
金发的网瘾少年(促进发展=进发=金发)在用键盘打游戏，妈妈在旁边喂饭(范围=喂饭)。

包容审慎
桩子：咖啡杯
一包咖啡豆溶解在咖啡里(包容审慎=一包咖啡豆溶解)。

平等对待
桩子：显示器
显示器的两边不平，左边角比右边角高，用手扭动到左右持平(平等对待=用手扭动到左右持平)。

均衡保障
桩子：笔记本
笔记本上有均匀的横线。想象自己冬天在笔记本上写字的时候，戴手套保障自己不生冻疮(均衡保障=均匀横线+戴手套)。

协同监管
桩子：台灯
我斜着肩膀检查台灯哪里坏掉了(协同监营=协监=斜肩膀)。

社会共治
桩子：主机箱
我弓着背用手指按主机箱(共治=弓背指按)。

法律衔接
桩子：椅子
我坐在椅子上梳理自己的发际线(法律衔接=法衔=发际线)



大型网站软件系统特点
桩子：计算机
用计算机登录网站(大型网站)观看奥运会跳水比赛。计算机的鼠标下面有软垫(软件=软垫)。计算机开机会显示 WINDOWS 操作系统(系统)跳水会溅起特别多水花，水花点点飞起来(特点=水花点点)。

高并发、高可用
桩子：跳水台
跳水台可以用来跳水(高可用)，两个人一并跳入水中有水花(高并发)。

海量数据
桩子：泳池
游泳池里蓝色的水很像大海。跳水运动员跳下来后，裁判会对他们的表现进行打分，如8.5分，这个分数是跳水的得分数据(海量数据)。

用户分布广泛、网络情况复杂
桩子：座位
座位上坐满了来自不同国家的观众(用户分布广泛)。一些座位上的观众接到电话听不清,跑出接电话，说明泳池馆内网络不好(网络情况复杂)

安全环境恶劣
桩子：栏杆
栏杆上都是倒刺，路过的观众很容易刮伤，这是安全环境恶劣的表现

需求快速变更，发布频繁;渐进式发展
桩子：扶手
运动员扶着扶手上岸，指着毛巾让教练拿过来，这时候突然摔倒，他的需求从要毛巾变成了要医药箱(需求快速变更)。受伤后，运动员拍照发布朋友圈一张接一张地发(发布频繁)。周围的医务人员迈着小碎步过来(渐进式发展;小碎步=渐进)，帮运动员上药。


```
